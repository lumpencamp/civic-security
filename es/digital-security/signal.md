# Guía: Mensajero de señales

**¿Por qué Signal?** Signal utiliza cifrado de extremo a extremo para todos los mensajes y llamadas, lo que significa que solo usted y el destinatario pueden leerlos. Es de código abierto y está administrado por una organización sin fines de lucro.

**Configuraciones clave de seguridad:**
1. **Bloqueo de registro:** Establezca un PIN para evitar que otros registren su número en un nuevo dispositivo. (Configuración > Cuenta > Bloqueo de registro)
2. **Bloqueo de pantalla:** Solicite una contraseña, PIN o datos biométricos para abrir la aplicación. (Configuración > Privacidad > Bloqueo de pantalla)
3. **Mensajes que desaparecen:** Configure los mensajes para que se eliminen automáticamente después de un cierto tiempo para conversaciones confidenciales. (Dentro de un chat > Configuración de chat > Mensajes que desaparecen)
4. **Desactivar vistas previas de enlaces:** evita que Signal obtenga una vista previa de los enlaces que envías. (Configuración > Privacidad > Generar vistas previas de enlaces)
