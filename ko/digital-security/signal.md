# 가이드: 시그널 메신저

**Signal을 선택해야 하는 이유** Signal은 모든 메시지와 통화에 엔드투엔드 암호화를 사용합니다. 즉, 귀하와 수신자만 읽을 수 있습니다. 오픈 소스이며 비영리 단체에서 운영됩니다.

**보안을 위한 주요 설정:**
1. **등록 잠금:** 다른 사람이 새 장치에 귀하의 번호를 등록하지 못하도록 PIN을 설정합니다. (설정 > 계정 > 등록 잠금)
2. **화면 잠금:** 앱을 열려면 비밀번호, PIN 또는 생체 인식이 필요합니다. (설정 > 개인정보 보호 > 화면 잠금)
3. **사라지는 메시지:** 민감한 대화의 경우 특정 시간이 지나면 자동으로 삭제되도록 메시지를 설정합니다. (채팅 내 > 채팅 설정 > 메시지 사라지기)
4. **링크 미리보기 끄기:** Signal이 보내는 링크의 미리보기를 가져오는 것을 방지합니다. (설정 > 개인정보 보호 > 링크 미리보기 생성)
