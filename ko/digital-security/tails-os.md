# 가이드: 테일즈 OS

Tails는 감시 및 검열로부터 보호하는 휴대용 운영 체제입니다. Tor 네트워크를 통해 모든 인터넷 연결을 강제하고 이를 사용하는 컴퓨터에 흔적을 남기지 않습니다.

**사용 방법:**
1. **다운로드:** Tails 웹사이트에서 공식 이미지를 다운로드하세요.
2. **부팅 가능한 USB 만들기:** balenaEtcher와 같은 도구를 사용하여 Tails 이미지를 USB 드라이브(최소 8GB)에 플래시합니다.
3. **USB에서 부팅:** 컴퓨터를 다시 시작하고 기본 OS 대신 USB 드라이브에서 부팅합니다.
4. **영구 저장소(선택 사항):** USB에 암호화된 영구 볼륨을 생성하여 재부팅 시 파일, 설정 및 소프트웨어를 저장할 수 있습니다.

_최종 업데이트: 2026_
