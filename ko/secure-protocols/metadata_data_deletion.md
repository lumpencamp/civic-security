# 메타데이터 스크러빙 및 안전한 데이터 삭제

이 가이드에서는 파일을 공유하기 전에 숨겨진 데이터를 정리하는 방법과 포렌식 도구로 복구할 수 없도록 데이터를 영구적으로 삭제하는 방법을 다룹니다.

## 메타데이터란 무엇입니까?

메타데이터는 "데이터에 관한 데이터"입니다. 파일(사진, 동영상, 문서) 내부에 숨겨져 있는 정보입니다.

* **사진/동영상(EXIF 데이터):** 스마트폰으로 촬영한 사진에는 촬영 장소, 시간 및 날짜, 휴대폰 모델, 심지어 카메라가 향하고 있는 방향에 대한 정확한 GPS 좌표가 포함되는 경우가 많습니다.
* **문서(Word, PDF):** 여기에는 작성자 이름, 조직 이름, 사용된 소프트웨어 및 작성 날짜가 포함되는 경우가 많습니다.

메타데이터를 삭제하지 않고 온라인에 항의 사진을 게시하거나 기획 문서를 기자에게 보내면 실수로 귀하의 신원이나 위치가 공개될 수 있습니다.

### 메타데이터 스크러빙을 위한 도구

* **모바일(Android):** **ExifEraser** 또는 **Scrambled Exif**를 사용하세요. 사진을 앱에 공유하면 메타데이터가 제거된 다음 깨끗한 사진이 공유됩니다.
* **모바일(iOS):** **메타포**를 사용하세요.
* **데스크톱(Windows/Mac/Linux):** **MAT2**(Metadata Anonymisation Toolkit)가 표준입니다. 명령줄을 통해 또는 Tor 브라우저용 플러그인으로 작동합니다.
* **문서:** PDF 및 사무용 문서의 경우 **Dangerzone**을 사용하세요. 문서를 픽셀로 렌더링하고 그 반대로 렌더링하여 안전한 PDF로 변환하여 모든 메타데이터와 잠재적인 맬웨어를 완전히 파괴합니다.

## 안전한 데이터 삭제

파일을 "휴지통"으로 끌어서 비워도 컴퓨터는 실제로 파일을 지우지 않습니다. 단지 공간을 "사용 가능"으로 표시할 뿐입니다. 법의학 소프트웨어(예: 경찰이 사용하는 Cellebrite)는 이러한 "삭제된" 파일을 쉽게 복구할 수 있습니다.

### 전체 디스크 암호화(FDE)의 역할

삭제된 데이터를 보호하는 가장 효과적인 방법은 전체 장치를 암호화(전체 디스크 암호화)하는 것입니다.
* 하드 드라이브가 암호화되어 있고 전원이 꺼진 상태에서 경찰이 이를 압수하는 경우 활성 데이터든 삭제된 데이터든 *모든* 데이터를 읽을 수 없습니다.
* 최신 스마트폰(iOS/Android) 및 최신 운영 체제(macOS FileVault, Windows BitLocker, Linux LUKS)에서는 FDE를 사용합니다. 켜져 있는지 확인하세요.

### 보안 삭제 도구

컴퓨터를 계속 사용하는 동안 특정 파일이 삭제되었는지 확인해야 하는 경우:

* **데스크톱:** **BleachBit**을 사용하세요. 임의의 데이터로 하드 드라이브의 여유 공간을 덮어쓰면 복구가 불가능해집니다.
* **SSD에 대한 참고 사항:** 솔리드 스테이트 드라이브(SSD)는 기존 자기 하드 드라이브와 다르게 데이터를 처리합니다. SSD의 특정 파일을 반복적으로 덮어쓰는 것은 신뢰할 수 없으며 드라이브 성능을 저하시킵니다. 전체 디스크 암호화를 사용하는 것은 최신 하드웨어에 대한 가장 안전한 접근 방식입니다.
